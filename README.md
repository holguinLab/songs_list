### ⚡ Notas Esenciales  
- **Clonar el repo:**  
  ```bash
  git clone https://github.com/holguinLab/songs_list.git
  cd song-list
  ```  
- **Instalar dependencias:**  
  ```bash
  npm install
  ```  
- **Subir cambios al repositorio:**  
  ```bash
  git add .
  git commit -m "Descripción de cambios"
  git push origin main
  ```
